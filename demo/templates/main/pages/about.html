Demonstrate

This site aims to...

demonstrate modern solutions in web applications world,

show how using most famous python web frameworks may help in fast application development,

exemplify how to optimize and speed up things with multithreading and multiprocessing,

demonstrate how different services work together under the hood of one application,

... and simply to utilize knowledge and skills.

Explain

This site consists of two parts:

- the first one is well animated and nice presentation - see pages About, Architecture, Features, etc.

- the second part demonstrates concrete technologies and has different UI - see Dashboard page.

And everything related to demoing tech stack you will find there:

- Django-based pages which use pure class-based views and Django templates, with simple visualization via tables and charts,

- DRF-based API which interacts with PostgreSQL database,

- FastaAPI application with its own API endpoints and logic,

- In its turn FastAPI processes user requests using different ways: via asyncio, multithreading, multiprocessing and Celery,

- Event-driven approach is represented by event Producer in FastAPI view, Kafka as message broker and Consumer service.

- And finally there are some other useful pages like Django Admin site, Adminer DBM, Silk profiler, JupiterLab, and Swagger-based API docs for both DRF and FastAPI endpoints.

Visualize

Usually such things like application structure, its services and deployment remain under the hood without any explanation.

And this demo site shows, explains and visualizes such sometimes implicit things like:

- application architecture,

- different approaches to parallelize user tasks,

- services orchestration and deployment

via schemas, charts and tables.
